simple-spring4-webapp-gradle
============================

Simple Spring 4 MVC web app using Gradle

This is probably the simplest possible Spring 4 MVC web app using gradle.

.
├── build.gradle
└── src 
    ├── main
    │   ├── java
    │   │   └── rooney
    │   │       └── james
    │   │           ├── config
    │   │           │   ├── WebConfig.java
    │   │           │   └── WebInitializer.java
    │   │           └── controller
    │   │               └── HomeController.java
    │   ├── resources
    │   └── webapp
    │       └── WEB-INF
    │           └── views
    │               └── home.jsp
    └── test
        └── java
